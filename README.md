# Site Psicopedagoga Rafaela Soares

Um site profissional e responsivo para uma psicopedagoga clínica em Jaguaribe, Ceará. O projeto apresenta serviços, informações de contato e integração com WhatsApp para agendamentos.

## 🎯 Sobre o Projeto

Site moderno desenvolvido para **Rafaela Soares**, psicopedagoga clínica especializada em:
- Dificuldades de aprendizagem
- Desenvolvimento cognitivo
- Acompanhamento psicopedagógico infantil
- Atendimento presencial e online

**Localização**: Jaguaribe, Ceará - Brasil  
**URL**: https://site-psicopedagogia-rafaela-soares-nu.vercel.app

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica com SEO otimizado
- **CSS3 / Tailwind CSS** - Estilização responsiva via CDN
- **JavaScript** - Interatividade e integração com WhatsApp
- **AOS (Animate On Scroll)** - Animações ao scroll
- **Google Fonts** - Tipografia profissional:
  - Playfair Display (títulos)
  - Inter (corpo do texto)
  - Dancing Script (elementos decorativos)

## 📁 Estrutura do Projeto

```
frontend/
├── index.html          # Página principal com toda a estrutura SEO
├── index.js           # Scripts de interatividade
└── img/               # Pasta para imagens/assets
```
